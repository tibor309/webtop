# 🖥️ Fedora XFCE Spin
The [Fedora Xfce spin][fedora-xfce] showcases the Xfce desktop, which aims to be fast and lightweight, while still being visually appealing and user friendly.

![fedora-xfce](https://github.com/user-attachments/assets/2be4133d-f8cc-4546-9458-3809dbd12b81)

## Setup
> [!WARNING]
> This desktop will be removed soon! Existing builds of the image are going to be still available on the GitHub Container Registry. Please migrate over to the [linuxserver.io][lswebtop] image or use another desktop.

To use this desktop, change the docker image tag to `fedora-xfce`. For setup options and configuration, check the [main page][main].
